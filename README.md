# deep-learning-stock-prediction-portfolio-optimization
A deep learning-driven stock investment planning system that forecasts NSE stock prices sector-wise using CNN, LSTM, Hybrid, and Transformer models. Includes comparative performance analysis, 5-day price forecasting, and automated visualization for investment insights.
